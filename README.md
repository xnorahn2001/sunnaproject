### لوحات التحكم
- ✅ لوحة تحكم المصمم
- ✅ لوحة تحكم الشركة

## التقنيات المستخدمة

- **HTML5**: لبناء الصفحات
- **CSS3**: للأنماط والتصميم
- **JavaScript**: للتفاعلات والوظائف
- **LocalStorage**: لتخزين بيانات المستخدم مؤقتاً

## كيفية الاستخدام

### 1. فتح الموقع
1. استخرج ملفات المشروع
2. افتح ملف `index.html` في المتصفح

### 2. التنقل بين الصفحات
- استخدم القائمة العلوية للتنقل
- اضغط على شعار "منصة" للعودة للصفحة الرئيسية

### 3. تسجيل حساب جديد
1. اضغط على "إنشاء حساب"
2. اختر نوع حسابك (مصمم، شركة، إلخ)
3. ملء البيانات المطلوبة
4. اضغط "إنشاء حساب"

### 4. تسجيل الدخول
1. اضغط على "تسجيل الدخول"
2. أدخل بريدك الإلكتروني وكلمة المرور
3. اضغط "تسجيل الدخول"
4. سيتم نقلك لصفحة لوحة التحكم

## البيانات المؤقتة

يتم تخزين بيانات المستخدم مؤقتاً باستخدام `localStorage`:
- `userLoggedIn`: حالة تسجيل الدخول
- `userName`: اسم المستخدم
- `userEmail`: البريد الإلكتروني
- `userType`: نو�� الحساب

## التطوير والتخصيص

### إضافة صفحة جديدة
1. أنشئ ملف HTML جديد في مجلد `pages/`
2. استخدم نفس الهيكل الأساسي (navbar وfooter)
3. أضف الروابط في ملفات الملاحة

### تعديل الألوان والأنماط
عدّل المتغيرات في `css/style.css`:
```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #9333ea;
    /* ... متغيرات أخرى ... */
}
```

### إضافة وظائف جديدة
أضف الوظائف في الملفات المناسبة:
- `js/main.js` - للوظائف العامة
- `js/dashboard.js` - لوظائف لوحات التحكم
- `js/form-validation.js` - للتحقق من النماذج

## ملاحظات مهمة

- جميع الملفات معربة بالكامل (أسماء الدوال والتعليقات)
- الموقع يدعم اللغة العربية بشكل كامل (RTL)
- النماذج تحتوي على تحقق من البيانات من جانب المتصفح
- يمكن توسيع المشروع بإضافة backend و database
