# Assets Directory

## محتويات المجلد

### images/ - الصور
ضع هنا جميع الصور المستخدمة في الموقع:
- شعارات
- خلفيات
- بنرات
- صور المنتجات

### icons/ - الأيقونات
ضع هنا الأيقونات بصيغ مختلفة:
- SVG icons
- PNG icons
- Font icons

### uploads/ - الملفات المرفوعة
مجلد لتخزين الملفات المرفوعة من المستخدمين:
- تصاميم المستخدمين
- البورتفوليوهات
- الملفات المرفقة

### data/ - بيانات مؤقتة
ملفات JSON للبيانات المؤقتة (Mock Data):
- بيانات المشاريع
- بيانات المستخدمين
- بيانات الخدمات
